## 아카이브 디렉터리 생성
- WAL 아카이브 파일을 저장할 디렉터리를 만든다
```bash
[root@orcl ~]# mkdir -p /archive/pg15 # WAL 아카리읍 파일을 저장할 디렉터리를 생성한다
[root@orcl ~]# chown postgres:postgres /archive/pg15 # /archive/pg15 디렉터리 소유자를 postgres로 변경한다
[root@orcl ~]# chmod 700 /archive/pg15 # /archive/pg15 접근 권한을 소유자만 가능하도록 설정한다
```

## 현재 설경을 변경한다
- wal_level = replica : 로그에 기록할 정보의 양을 아카이브/복제에 필요한 수준으로 설정한다
- archive_mode = on : 아카이브 기능을 활성화한다
- archive_command = 'cp %p /archive/pg15/%f : WAL 세그먼트가 전환될 때 %p(원본 경로)를 /archive/pg15/%f(파일명)로 복사하도록 설정한다
```bash
# 현재 설정 조회
-bash-4.2$ grep -E 'archive_mode|archive_command|wal_level' /var/lib/pgsql/15/data/postgresql.conf
#wal_level = replica                    # minimal, replica, or logical
#archive_mode = off             # enables archiving; off, on, or always
                                # (empty string indicates archive_command should
#archive_command = ''           # command to use to archive a logfile segment

-bash-4.2$ sed -i "s/^#*wal_level.*/wal_level = replica/" /var/lib/pgsql/15/data/postgresql.conf
-bash-4.2$ grep -E 'archive_mode|archive_command|wal_level' /var/lib/pgsql/15/data/postgresql.conf
wal_level = replica # 주석 제거
#archive_mode = off             # enables archiving; off, on, or always
                                # (empty string indicates archive_command should
#archive_command = ''           # command to use to archive a logfile segment

# 아카이브 모드 on으로 변경
-bash-4.2$ sed -i "s/^#*archive_mode.*/archive_mode = on/" /var/lib/pgsql/15/data/postgresql.conf
-bash-4.2$ sed -i "s|^#*archive_command.*|archive_command = 'cp %p /archive/pg15/%f'|" /var/lib/pgsql/15/data/postgresql.conf

-bash-4.2$ grep -E 'wal_level|archive_mode|archive_command' /var/lib/pgsql/15/data/postgresql.conf
wal_level = replica
archive_mode = on  
                                # (empty string indicates archive_command should
archive_command = 'cp %p /archive/pg15/%f'
```

## 파일 블록 손상
```bash
-bash-4.2$ dd if=/dev/urandom of="$REAL_FILE" bs=8192 count=1 conv=notrunc
1+0 records in
1+0 records out
8192 bytes (8.2 kB) copied, 0.000483406 s, 16.9 MB/s
```
```
if=/dev/urandom : 랜덤 데이터를 읽음
of=$REAL_FILE   : 테이블 물리 파일에 기록
bs=8192         : PostgreSQL 기본 페이지 크기 8KB
count=1         : 첫 번째 블록 1개만 손상
conv=notrunc    : 파일 크기는 유지하고 내용만 덮어씀
```
