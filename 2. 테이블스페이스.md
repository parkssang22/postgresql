## 디렉토리 생성 및 권한 부여
```bash
[root@localhost ~]# mkdir -p /data/pg_tbs/users
[root@localhost ~]# chown postgres:postgres /data/pg_tbs/users
[root@localhost ~]# chmod 700 /data/pg_tbs/users
```

## 테이블 생성할 때 테이블스페이스 지정 가능
```bash
CREATE TABLE emp (
    empno     NUMERIC(4)    NOT NULL,
    ename     VARCHAR(10),
    job       VARCHAR(9),
    mgr       NUMERIC(4),
    hiredate  DATE,
    sal       NUMERIC(7,2),
    comm      NUMERIC(7,2),
    deptno    NUMERIC(2),
    CONSTRAINT pk_emp PRIMARY KEY (empno)
) TABLESPACE pg_default;
```
