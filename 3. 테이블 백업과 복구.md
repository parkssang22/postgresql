## vi .bash_profile로 열고 수정하고 source .bash_profile
```bash
-bash-4.2$ cat .bash_profile
[ -f /etc/profile ] && source /etc/profile
PGDATA=/var/lib/pgsql/15/data
export PGDATA
# If you want to customize your settings,
# Use the file below. This is not overridden
# by the RPMS.
[ -f /var/lib/pgsql/.pgsql_profile ] && source /var/lib/pgsql/.pgsql_profile
# PostgreSQL 15 경로를 앞쪽에 추가
export PATH=/usr/pgsql-15/bin:$PATH
export PGDATA=/var/lib/pgsql/15/data
export PGHOST=localhost
```

## scott의 테이블 백업
```bash
-bash-4.2$ pg_dump -U scott -d testdb -t scott.emp -t scott.dept -Fc -f backup.dump
암호:
-bash-4.2$
-bash-4.2$ ls -l backup.dump
-rw-r--r--. 1 postgres postgres 2283  4월 22 15:58 backup.dump
-bash-4.2$ file backup.dump
backup.dump: PostgreSQL custom database dump - v1.14-0
-bash-4.2$ echo $?
0
```
